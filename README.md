# 3JS Teapot

A mesmerizing particle-based teapot rendered using Three.js, featuring dynamic glow effects and post-processing shaders.

## Features

- **Particle Teapot**: The classic Utah teapot geometry converted into a point cloud with thousands of particles.
- **Dynamic Coloring**: Gradient color palette with sparkling effects on random particles.
- **Post-Processing Effects**: Includes bloom, film grain, RGB shift for a stylized look.
- **Interactive Controls**: Orbit controls for rotating and zooming the camera.
- **Responsive Design**: Adapts to window resizing.

## How to Run

Simply open `index.html` in a modern web browser. No server required.

## Technologies Used

- [Three.js](https://threejs.org/) v0.162.0 - 3D graphics library
- Post-processing effects from Three.js examples
- HTML5 Canvas and WebGL
